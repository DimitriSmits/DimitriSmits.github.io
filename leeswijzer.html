1. Projectopdracht

1.1 Context

CY2 is gespecialiseerd in chatbots. Voornamelijk voor universiteiten en scholen waarbij studenten worden geholpen met vraagstukken. Om de chatbot nog beter te maken wordt er een feedback module gemaakt waar mensen feedback kunnen geven over de ervaringen. Doormiddel van deze feedback komt cy2 sneller aan nieuwe inzichten, dit kan leiden tot veranderingen waardoor de kwaliteit van de bot beter wordt. Hierdoor wordt de chatbot ook gebruiksvriendelijker. Daarnaast wordt de feedback module zo gemaakt dat deze automatisch toegevoegd kan worden aan een bestaande chatbot. Deze mogelijkheid maakt het makkelijk voor bedrijven om alsnog de feedback module aan de chatbot toe te voegen.

1.2 Doel van het project

Dit project wordt gerealiseerd zodat het functioneren van de chatbot verbetert kan worden door de feedback die gebruikers achterlaten. Daarnaast wordt de feedback module zo gemaakt dat hij automatisch toegevoegd kan worden aan een chatbot, dit bespaart tijd voor Cy2 en de klant.
